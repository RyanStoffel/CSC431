Test Case 3: Simulated Collision:
=================================
Input 1: "collision_test"
Input 2: "SIMULATED_COLLISION" (injected via simulation)

MD5:
  Hash: beaf94407d710360689d3a1223d54598
  Inputs: [collision_test, SIMULATED_COLLISION]

SHA-1:
  Hash: 6c8038e184a735e0f25318b3b06335a9a84458ba
  Inputs: [collision_test, SIMULATED_COLLISION]

SHA-256:
  Hash: dd2b69d9efc44cffdcc2872dbbd893a5bfd856bc712b1106657ad3a56b9f1046
  Inputs: [collision_test, SIMULATED_COLLISION]

Collision Simulated: Yes
Collision Detected: Yes (all three algorithms)

Notes: Demonstrates the collision detection feature. The simulated collision injects "SIMULATED_COLLISION" into the first hash bucket, triggering a collision report across MD5, SHA-1, and SHA-256. Real-world MD5 collisions exist (Wang et al.), but SHA-256 collisions have not been demonstrated.
